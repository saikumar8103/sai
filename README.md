Differentiable Time-Varying IIR Filtering for Real-Time Speech Denoising
1. Introduction

Speech denoising is a fundamental problem in audio signal processing, with applications in telecommunications, hearing aids, voice assistants, and real-time communication systems. Traditional denoising techniques often rely on fixed filters or non-adaptive methods that struggle in dynamic acoustic environments where noise characteristics change over time.

Infinite Impulse Response (IIR) filters are widely used due to their computational efficiency and ability to model recursive systems. However, classical IIR filters are typically static and not easily adaptable in real time. Moreover, their parameters are not inherently differentiable in a way that integrates seamlessly with modern machine learning frameworks.

This project introduces Differentiable Time-Varying IIR Filtering, a novel approach that combines:

Classical signal processing (IIR filtering)
Deep learning (neural parameter estimation)
Differentiable programming (end-to-end training)

The core idea is to design an IIR filter whose coefficients change over time and are generated by a neural network. By making the filtering process differentiable, the system can be trained using gradient-based optimization to minimize noise while preserving speech quality.

2.4 Differentiable Filtering

To enable training:

The recursive IIR operation is implemented in a differentiable manner
Gradients are propagated through time (similar to RNNs)

Techniques used:

Backpropagation Through Time (BPTT)
Stability constraints on coefficients
Gradient clipping
2.5 Stability Constraints

To ensure filter stability:

Poles must lie inside the unit circle
Parameterization methods:
Reflection coefficients
Sigmoid/tanh mapping
Spectral normalization
